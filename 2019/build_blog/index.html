github + hugo 搭建博客

Written by 运动书生 with ♥ on June 25, 2019 in

正如你看到的，本博客正是用github + hugo搭建，搭建过程主要参考文档

准备工作

  1. github 申请账号，创建库
  2. hugo 本地安装
  3. hugo 生成的静态网页上传到github

1. github

创建一个新的仓库，库名以github.io结尾

github pages设置

1为访问博客的地址

2处选择master分支即可

2. hugo安装

如果mac上已安装Homebrew，直接运行 brew install hugo

在电脑选取一个位置存放博客内容的地方 hugo new site 目录名称

在目录下会有themes目录，这里用于安装主题 主题的概念初次使用可能不好理解，主要用于你的博客展示形式，本博客使用的是 LeaveIt

在博客目录下执行如下命令 cd themes & git clone https://github.com/liuzc/LeaveIt.git

后面主要是修改根目录下的config.toml 文件，可直接把themes/LeaveIt/exampleSite/config.toml 拷贝过来修改

使用hugo new about.md 可以在context目录下生成对应的markdown文件

3. 发布到github

在根目录执行hugo，会生成public文件

首先执行命令 cd public 进入到 public 文件夹中 然后执行

git init

git remote add origin https://github.com/库

git add -A

git commit -m "[介绍，随便写点什么，比如日期]"

git push -u origin master

后续有文章更新 需要rm -rf public目录，重新上传

git init

git remote add origin https://github.com/库

git add -A

git commit -m "[介绍，随便写点什么，比如日期]"

git push -f https://github.com/库 master

本地测试hugo server --buildDrafts -w buildDrafts 会把draft=true的也生成网页，-w是内容变化及时变更
